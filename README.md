# Day-95-of-100-days-of-python
🚀 Day 95 of #100DaysOfPython

Today I built **BrewScout** ☕🌍 — a web app that helps you find cafés based on your location using the **Open Brewery DB API**.

You simply enter a location, and the app fetches and displays nearby places in a clean **card-based layout**.

🔧 What I worked on:
• Integrating a real-world **public API**
• Fetching and displaying location-based data
• Structuring results in a **user-friendly UI (box/card format)**
• Connecting backend logic with frontend rendering

💡 This project showed me how APIs can power real-world applications and deliver useful data to users instantly.

Building apps that people can actually use — that’s the goal 🚀
